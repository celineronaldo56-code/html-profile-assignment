Question 9

What is the purpose of the class attribute in HTML

Answer 9

The class attribute in HTML is used to define a class name in a style sheet. It allows you to apply CSS styles to multiple elements on a web page by assigning them the same class name. This helps in maintaining consistency and makes it easier to manage and update styles across the website.


Question 10

Can multiple HTML elements share the same class name? Explain how this can be useful in web development.

Answer 10

Yes, multiple HTML elements can share the same class name. This is useful in web development because it allows developers to apply the same styles and behaviors to a group of elements without having to write separate CSS rules for each one. It promotes code reusability, simplifies maintenance, and ensures a consistent look and feel across the website.
